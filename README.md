# 📊 Ecommerce Sales Analysis Using Power BI

## 📌 Project Overview

This project presents an interactive Ecommerce Sales Dashboard developed using Power BI.  
The dashboard provides insights into:

- Sales Performance
- Profit Analysis
- Customer Trends
- Regional Analysis
- Product Category Performance

The goal of this project is to transform raw ecommerce data into meaningful business insights using visualization techniques.

---

# 🛠 Tools Used

- Power BI
- Excel / CSV Dataset
- Data Cleaning
- DAX Measures
- Data Visualization

---

# 📂 Dataset

The dataset contains ecommerce sales information including:

- Order Details
- Product Categories
- Sales
- Profit
- Customer Segments
- Regions
